# 🎨 LuminaBridge 前端设计方案

**版本**: v1.0  
**创建时间**: 2026-03-22  
**定位**: Web 多租户管理面板

---

## 📋 设计目标

结合 one-api、new-api、burncloud 的优点，打造：
- ✅ **用户满意度高** - 现代化 UI、易用性强
- ✅ **运维效率高** - 自动化、可视化、批量操作
- ✅ **多租户支持** - 完整的租户隔离和权限管理
- ✅ **前后端交互流畅** - API 设计合理、响应快速

---

## 🎯 技术栈选择

### 前端框架

| 组件 | 技术选型 | 理由 |
|------|----------|------|
| **核心框架** | React 18 + TypeScript | 生态丰富、类型安全、性能好 |
| **UI 组件库** | Ant Design 5.x / MUI | 企业级组件、多主题支持 |
| **状态管理** | Zustand / Redux Toolkit | 轻量、易用、DevTools 支持 |
| **路由** | React Router v6 | 成熟稳定、支持嵌套路由 |
| **HTTP 客户端** | Axios + React Query | 请求拦截、缓存、重试机制 |
| **图表库** | ECharts / Recharts | 丰富的图表类型、性能好 |
| **构建工具** | Vite | 快速开发、热更新、打包优化 |
| **样式方案** | Tailwind CSS + CSS Modules | 原子化 CSS、局部作用域 |

### 后端 API

| 组件 | 技术选型 | 理由 |
|------|----------|------|
| **Web 框架** | Axum (Rust) | 高性能、类型安全、生态好 |
| **认证** | JWT + OAuth 2.0 | 无状态、支持多提供商 |
| **数据库** | PostgreSQL / SQLite | 灵活部署、性能好 |
| **缓存** | Redis | 高性能、支持分布式 |
| **实时通信** | WebSocket | 实时监控、推送通知 |

## 🔌 API 接口设计

### 认证相关

```typescript
// 登录
POST /api/v1/auth/login
{
  "email": "user@example.com",
  "password": "password123"
}

// OAuth 登录
GET /api/v1/auth/oauth/github
GET /api/v1/auth/oauth/discord

// 刷新 Token
POST /api/v1/auth/refresh
{
  "refresh_token": "xxx"
}

// 登出
POST /api/v1/auth/logout
```

### 租户相关

```typescript
// 获取当前租户信息
GET /api/v1/tenant

// 更新租户配置
PUT /api/v1/tenant
{
  "name": "新租户名称",
  "settings": { ... }
}

// 获取租户用量统计
GET /api/v1/tenant/usage?start=2026-03-01&end=2026-03-31
```

### 渠道相关

```typescript
// 获取渠道列表
GET /api/v1/channels?page=1&pageSize=20&group=default

// 创建渠道
POST /api/v1/channels
{
  "name": "OpenAI-1",
  "type": "openai",
  "key": "sk-xxx",
  "base_url": "https://api.openai.com/v1",
  "models": ["gpt-4", "gpt-3.5-turbo"],
  "weight": 10
}

// 更新渠道
PUT /api/v1/channels/:id

// 删除渠道
DELETE /api/v1/channels/:id

// 测试渠道
POST /api/v1/channels/:id/test

// 批量操作
POST /api/v1/channels/batch
{
  "action": "enable", // enable/disable/delete
  "ids": [1, 2, 3]
}
```

### 令牌相关

```typescript
// 获取令牌列表
GET /api/v1/tokens?page=1&pageSize=20

// 创建令牌
POST /api/v1/tokens
{
  "name": "My API Key",
  "quota": 1000000,
  "expired_time": 1711900800,
  "allowed_models": ["gpt-4", "gpt-3.5-turbo"]
}

// 删除令牌
DELETE /api/v1/tokens/:id

// 更新令牌额度
PATCH /api/v1/tokens/:id/quota
{
  "quota": 2000000
}
```

### 统计相关

```typescript
// 获取实时统计
GET /api/v1/stats/realtime

// 获取用量统计
GET /api/v1/stats/usage?start=2026-03-01&end=2026-03-31&group_by=day

// 获取渠道统计
GET /api/v1/stats/channels?start=2026-03-01&end=2026-03-31

// 获取模型统计
GET /api/v1/stats/models?start=2026-03-01&end=2026-03-31
```
